The directory permissions are wrong. Directories need execute permission:

---

## Step 1: Fix Directory Permissions

```bash
sudo chmod -R 755 /etc/docker/certs.d/
```

---

## Step 2: Verify

```bash
ls -la /etc/docker/certs.d/36.255.69.72:4433/
```

Should show:
```
-rw-r--r-- ca.crt
```

---

## Step 3: Restart Docker

```bash
sudo systemctl restart docker
```

---

## Step 4: Test Login

```bash
docker login -u admin -p 786Shuvooo 36.255.69.72:4433
```

Should show:
```
Login Succeeded
```

---

If still fails, use the insecure registry approach:

```bash
sudo nano /etc/docker/daemon.json
```

Add:
```json
{
  "insecure-registries": ["36.255.69.72:4433"]
}
```

Restart:
```bash
sudo systemctl restart docker

docker login -u admin -p 786Shuvooo 36.255.69.72:4433
```
